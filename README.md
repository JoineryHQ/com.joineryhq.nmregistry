# nmregistry

## CiviCRM extension: custom features for New Mexico Respite Providers registry.

The extension is licensed under [GPL-3.0](LICENSE.txt).

## Configuration
CiviCRM > Administer > NM Registry

## Requires
Extension: org.civicoop.emailapi
